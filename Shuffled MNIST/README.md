# OWM
Code for **Shuffled MNIST** in paper *[Continual Learning of Context-dependent Processing in Neural Networks](https://arxiv.org/abs/1810.01256)*

## Requirements:

- Linux: Ubuntu 16.04

- cuda9.0 & cudnn6.0

- Python 3.5.4

- torch 0.3.0 (pytorch)

- torchvision 0.2.0

- numpy 1.15.1

- scipy 1.0.0
